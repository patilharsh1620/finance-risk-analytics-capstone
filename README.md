## 📌 Project Overview

Investment decisions should balance **risk and return**. This project
analyzes the historical performance of **24 NYSE-listed companies**
across multiple sectors over a 10-year period to identify suitable
investment portfolios for investors with different financial goals.

Using **Python** for financial analysis and **Tableau** for interactive
dashboards, the project evaluates stock performance using various
financial metrics and recommends optimized portfolios.

------------------------------------------------------------------------

## 🎯 Business Problem

A financial advisory firm wants to recommend investment portfolios to
two clients.

### Client 1

-   Investment: **\$500,000**
-   Goal: Double investment in 5 years
-   Risk Preference: **Low Risk**

### Client 2

-   Investment: **\$1,000,000**
-   Goal: Maximum possible returns
-   Risk Preference: **High Risk**
